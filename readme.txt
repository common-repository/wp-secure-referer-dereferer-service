Automatically changes all external links on the blog to redirect through various anonymization services,
used to hide the source (referer) of traffic.

Secure-Referrer.biz is your safer Anonymous redirection service.
Prevents a site from appearing in the server logs of referred pages as referrer.

Webmasters can use this tool to prevent their site from appearing in the backlink statistics of another
webpage and server logs of referred pages as referrer. The operators of the referred pages cannot see
where their visitors come from any more. Secure-Referrer.biz provides a slick, quality derefering service for free.
No ads, interstitials or popups. We promise.

When using Secure-Referrer.biz you don't link directly to the external target web page but to Secure-Referrer.biz
which redirects your users to the desired page then.
So the target page doesn't get to know that the user actually came from your site.

Secure Referer removes the referer (referrer) from all external links on your blog.
It does this by converting all outgoing links on your blog using the Secure-Referrer.biz service and effectively
hides your site as origin of the referred link. This option can be activated in the Secure Referer Settings page.

German:

Das WordPress Referer Plugin wandelt automatisch alle externen Links auf der Webseite (Blog, Forum)
durch Unseren Anonymisierungsdienst um, und somit wird die Quelle / Herkunft (der Referer)
des Verkehrs umgeleitet und somit verschleiert / verborgen.

Secure-Referrer.biz ist Ihr sicherer Anonymer Umleitungsdienst. Unser Service verhindert,
dass die herkunft Webseite (Quelle) in den Server-Logs von der Ziel Webseite als Referrer erscheint.

Webmaster können dieses Tool verwenden, um zu verhindern, dass ihre Website in den Backlink Statistiken
von einer anderen Webseiten und den Server-Logs von Seiten als Referrer auftauchen.
Die Betreiber der verlinkten Seiten können nicht mehr sehen, woher ihre Besucher kommen. Secure-Referrer.biz
liefert einen Anonymen Hide Referer Service kostenlos, sicher und ohne sehr aufdringliche Werbung (Popups).

Beim Verwenden von Secure-Referrer.biz verlinken Sie nicht direkt auf die externe Ziel-Webseite, sondern
auf Secure-Referrer.biz, und durch eine Weiterleitungen kommen die Benutzer auf die gewünschte Seite.
So kann die Zielseite nicht zu wissen, dass der Benutzer tatsächlich von Ihrer Webseite stammt.

Secure Referer entfernt den Referer (die Herkumft Webseite) von allen externen Links auf Ihrem Blog.
Das funktioniert, indem man alle ausgehenden Links auf Ihrem Blog mit dem Secure-Referrer.biz Service
umwandelt, und versteckt Ihre Website als Ursprung der genannten Links effektiv.
Diese Option kann in den WP-Secure Referer Plugin Einstellungen aktiviert werden.

== Installation ==

1. Upload the plugin directory (wp-secure-referer-dereferer-service) to wp-content/plugins directory
2. Activate the plugin
3. Access the Plugin Options from the Options->Secure Referer menu
